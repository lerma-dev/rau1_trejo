# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## clonar repositorio
```bash
git clone https://github.com/lerma-dev/rau1_trejo.git
```

## Instalacion para usarlo 
```bash
npm install
```

## Ejecutarlo con los sig.
```bash
npm run dev
```

## Abrir la url  

http://localhost:5173
